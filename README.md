# NinetyDays
An app to get your fitness on track

## Introduction
NinetyDays is a personalized fitness-tracking app, based upon the [LeanGains](http://www.leangains.com/) protocol - a derivative of Intermittent Fasting

## Getting Started

```js
	> npm install
	> npm start
```
